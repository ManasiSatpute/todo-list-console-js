#  Todo List App (JavaScript)

A simple **console-based Todo List application** built using vanilla JavaScript.  
This project is beginner-friendly and demonstrates core JavaScript concepts.

---

##  Features

- Add new tasks
- View all tasks with index numbers
- Delete tasks by index
- Exit the application safely
- Handles invalid inputs gracefully

---

##  Technologies Used

- JavaScript (ES6)
- Browser Console
- `prompt()` for user input
- `console.log()` for output

---

##  Project Structure
```
├── problem.md
├── README.md
└── todo.js
```

---

##  How to Run

1. Copy the JavaScript code into a file named `todo.js`
2. Open your browser
3. Open **Developer Tools → Console**
4. Paste the code and press Enter
5. Follow the prompts to manage your todo list

---

##  Sample Commands

- `add` → Add a new task  
- `list` → Show all tasks  
- `delete` → Remove a task using its index  
- `quit` → Exit the application  

---

##  Learning Outcomes

- Working with arrays
- Using loops and conditional statements
- Handling user input
- Writing clean and readable JavaScript code
